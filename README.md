 AP-Chat-app-notepad-poker-game-
This repository contains three Java projects in one place:

- Chat App
- Notepad App
- Poker Game

These projects were created using Java to practice desktop application development, user interaction, object-oriented programming, and basic game logic.

 Projects

 1. Chat App

The Chat App is a simple Java messaging application. It provides a basic chat-style interface where users can type and view messages.

 Features

- Send messages
- View messages in a chat window
- Simple Java GUI
- Basic message handling

  2. Notepad App
The Notepad App is a simple Java text editor. It allows users to type, edit, and manage text in a notepad-style window.

  Features
-Write and edit text
-Simple text editor interface
-Java Swing GUI
-Basic notepad functionality


  3. Poker Game
The Poker Game is a console-based five-card draw poker game. The player starts with chips, places bets, receives cards, replaces selected cards, and competes against the dealer.

 Features
-Five-card draw poker
-Betting system
-Dealer opponent
-Card replacement
-Poker hand ranking
-Win, lose, and tie results
